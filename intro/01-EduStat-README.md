# EduStat

EduStat 是一个面向课堂教学场景的桌面端教学统计与辅助系统，聚焦教师在一门课程中的高频教学事务，提供学科管理、学生信息维护、课堂点名、智能分组、成绩分析与报告导出等功能，帮助教师提高教学管理效率与课堂组织效率。

## 项目简介

在真实教学过程中，教师往往需要同时处理课程配置、学生成绩登记、课堂点名、小组分配、阶段性成绩分析和报告导出等任务。这些工作过去通常分散在多个 Excel 表格和不同工具中，容易造成重复劳动和数据割裂。

EduStat 的目标是将这些教学事务整合到统一平台中，形成从课程建立、学生信息管理、课堂互动到成绩分析与结果输出的完整闭环。

## 核心功能

### 1. 学科管理

- 新增学科与课程编号自动生成
- 配置课程单元、权重与满分
- 展示已有课程卡片
- 删除课程及关联成绩数据

### 2. 学生信息管理

- 学生信息增删改查
- 按学号、姓名、班级搜索
- 支持 CSV / Excel 导入
- 支持 Excel 导出
- 支持按课程单元动态生成成绩表头
- 导出时可计算总分

### 3. 点名模块

- 随机点名
- 尽量不重复点名
- 仅抽取未点到学生
- 展示点名结果与历史记录

### 4. 组队模块

- 随机分组
- 强配弱平衡分组
- 每组人数设置
- 历史组队记录保存与恢复

### 5. 班级信息分析

- 学生总数、均分、最高分、最低分
- 成绩段分布柱状图
- 班级构成环形图
- 单元均分走势折线图
- 本地分析建议
- DeepSeek 辅助教学分析

### 6. 学期报告与成绩导出

- 导出 Markdown 学期报告
- 导出课程成绩表
- 按课程单元权重计算总分
- 未录入成绩按 0 分参与总分计算

## 从源码构建

### 1. 克隆项目

```bash
git clone https://github.com/baibinggyu/EduStat.git
cd EduStat
```

### 2. 使用 Qt Creator 打开

- 使用 Qt Creator 打开项目根目录下的 `CMakeLists.txt`
- 选择 Qt 6.10.2 MinGW 64-bit Kit

### 3. 编译运行

- 选择 Debug 或 Release
- 执行构建
- 运行 `EduStat.exe`

## 发布建议

Release 版本生成后，建议在可执行文件目录执行：

```bash
windeployqt EduStat.exe
```

确保以下模块随程序一起发布：

- Qt6Core
- Qt6Gui
- Qt6Widgets
- Qt6Sql
- Qt6Network
- Qt6Charts

如果需要打包单文件，可在完成依赖部署后使用其他打包工具进一步封装。

## 数据说明

系统核心数据围绕以下四类对象组织：

- `course`：课程
- `unit`：课程单元
- `student`：学生
- `score`：成绩

其中：

- 一个课程对应多个单元
- 一个学生在一个课程下对应多个单元成绩
- 总分根据单元权重和满分换算为 100 分制

## 学生导入格式

学生信息导入支持 CSV 或 Excel，推荐格式如下：

```csv
学号,班级,姓名,课堂表现,作业完成,阶段测验
20261001,学科教育1班,张三,86,92,88
20261002,学科教育1班,李四,90,,85
```

说明：

- 前三列必须为 `学号`、`班级`、`姓名`
- 后续列名必须与当前课程单元名称和顺序一致
- 空成绩允许导入
- 导出总分时，空成绩按 0 分处理

## DeepSeek 分析说明

系统支持通过 DeepSeek 接口生成教学分析建议。当前分析逻辑会结合：

- 课程整体统计
- 单元均分
- 成绩段分布
- 班级构成
- 学生样本数据

当 DeepSeek 不可用时，系统会自动回退到本地分析逻辑，保证基本功能可用。

如需启用 DeepSeek，请在项目运行目录中的 `init.json` 中正确配置 API Key。
