Features

核心能力

从课堂管理到语音识别，从视频教学到移动学习，覆盖教学全场景。

🖥️

桌面端教学管理

基于 Qt6 + QML + FluentUI 构建的现代化桌面应用，15 个功能页面覆盖班级信息、点名签到、组队算法、成绩管理、视频播放、公告发布等全流程。

📱

Flutter 移动端

完整的 Android 客户端，10 个页面实现登录、课程浏览、点名、公告、消息、视频播放等功能，通过 REST API 与后端服务通信。

🎬

视频播放与流媒体

桌面端内置 VideoPlayer（Qt Multimedia），支持倍速（0.25x–2.0x）、全屏、音量控制。后端基于 FastAPI 提供视频文件流媒体服务。

🤖

AI Agent 自主执行

桌面端内置 ReAct 循环 Agent（基于 ai-sdk-cpp），可调用本地工具链：操作 xlsx/CSV 文件、执行 Python 脚本、运行系统命令、分析数据。从"对话"到"执行"，Agent 直接帮你干活。

🎙️

语音识别 (ASR)

基于阿里 FunASR 框架的语音识别模块，支持音视频自动转写与字幕生成（sentence_timestamp 时间戳），可作为课堂录音转文字等功能的底层引擎。

🗄️

标准化数据架构

MariaDB 8 表设计，BIGINT 内部 ID + UUID 外部 ID 双层标识，涵盖用户、课程、成绩、视频播放记录等完整数据链路。

Subprojects

子项目一览

每个子项目均可独立构建和运行，共同组成完整的教育平台生态。

🖥️

EduStat_qml · 桌面教学管理端

Qt6/QMLFluentUIAI Agent桌面应用

基于 Microsoft Fluent Design 的现代化桌面客户端，最大亮点是内置 ReAct AI Agent——可自主操作本地 xlsx/CSV 文件、执行 Python 数据分析脚本、调用系统命令行，实现从"AI 聊天"到"AI 干活"的跨越。15 个功能页面：班级信息、点名签到、组队算法、学生信息、学科管理、开课申请、视频播放、课程资源、公告发布、倒计时、消息发送、AI 对话。内置 VideoPlayer 支持完整播放控制（倍速/全屏/字幕）。CMake 构建，C++20。

📱

Android · Flutter 移动客户端

Flutter/DartAndroid移动端

Flutter 构建的 Android 学习平台客户端。10 个页面：首页、课程列表、登录、学生信息、点名、公告、消息、个人中心、视频播放、倒计时。通过 AuthProvider 管理 JWT 登录态，ApiClient 统一封装 REST API 调用。支持 Linux 桌面构建。

🌐

edu_server · 后端服务

Python/FastAPIMariaDBJWT

完整的 REST API 后端，FastAPI + SQLAlchemy + MariaDB 技术栈。12 个路由模块：用户认证、课程管理、视频资源、播放记录、点名考勤、公告发布、消息系统、成绩管理、单元管理、文件上传、AI 对话。三级权限控制（学生/教师/管理员），bcrypt + JWT 鉴权。Nginx 反向代理部署。

🎙️

FunASR · 语音识别

PythonASRAI

基于阿里 FunASR 框架的语音识别模块，支持音频/视频文件的自动语音转写（ASR）。可作为教学场景中的字幕生成、课堂录音转文字等功能的底层引擎。

📱

edu_pe · QML 移动端原型

QML原型设计移动端

纯 QML 构建的手机端学习平台界面原型，390×844 手机尺寸。4 个底部 Tab（首页/课程/聊天/个人）+ 课程详情浮层。所有交互基于硬编码示例数据，用于快速验证移动端 UI 方案。

🗄️

Database · 数据架构设计

MariaDBSQLDDL

完整的数据库设计文档与 DDL 脚本。8 表架构：users、students、courses、course_members、units、scores、videos、play_records。utf8mb4 编码，InnoDB 引擎。DataGrip 项目配置随附。
