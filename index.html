education-statistics 是一个集桌面统计、视频播放、流媒体服务、移动端原型于一体的 综合性教育技术平台，致力于用技术赋能教学全流程。

Features

核心能力

从课堂管理到数据分析，从视频教学到移动学习，覆盖教学全场景。

📊

数据统计分析

成绩分布可视化、多维度统计报表，基于 Qt Charts 构建直观的数据看板。

🎬

视频播放与流媒体

支持倍速（0.25x–2.0x）、全屏、音量控制等标准播放体验，后端基于 FastAPI 提供流媒体服务。

📱

移动端学习原型

纯 QML 构建的手机端界面，390×844 类手机尺寸，4 个底部 Tab，模拟真实学习平台交互。

🤖

AI 智能辅助

集成 DeepSeek AI 接口（可选），实现智能分析与教学辅助能力。

🗄️

标准化数据架构

MariaDB 8 表设计，BIGINT 内部 ID + UUID 外部 ID 双层结构，涵盖用户、课程、成绩、视频全链路。

🔌

模块化架构

独立子项目，各自独立构建和运行，核心技术栈覆盖 C++/Qt、Python/FastAPI、QML、SQL。

Subprojects

子项目一览

每个子项目均可独立构建和运行，共同组成完整的教育平台生态。

📊

EduStat · 教学统计桌面端

Qt/C++桌面应用

教学统计与课堂管理桌面应用。基于 Qt Widgets + SQLite 构建，包含成绩分布柱状图、班级统计看板、SubjectCard/TeamCard 卡片视图等。支持 DeepSeek AI 接口（可选）。使用 CMake 构建，C++17 标准。

🎥

Video Player · 视频播放器

Qt6/C++桌面应用

独立视频播放器，基于 Qt Multimedia 构建。支持播放/暂停/停止、进度拖拽、倍速播放（0.25x–2.0x）、音量调节、全屏（双击切换）等完整播放功能。

🌐

Routing · 流媒体服务

Python/FastAPI后端服务

视频流媒体后端服务，基于 FastAPI + Uvicorn 构建。提供健康检查和视频流媒体端点。规划蓝图包含完整的后端体系：FastAPI + SQLAlchemy + MariaDB + JWT 认证。

📱

Edu PE · 移动端原型

QML移动端原型

纯 QML 构建的手机端学习平台界面原型，390×844 类手机尺寸。包含 4 个底部 Tab 导航 + 详情浮层，所有交互基于硬编码数据模拟。使用 #1296DB 主色调，卡片圆角 12px。

🗄️

Database · 数据架构

MariaDB数据库设计

标准化的数据库 DDL 和设计文档。8 表架构：用户、学生、课程、课程成员、单元、成绩、视频、播放记录。utf8mb4 编码，BIGINT 内部 ID + UUID 外部 ID 双层标识。
