# 开源代码与组件使用情况说明

本作品在开发过程中使用了部分通用开源框架和工具，主要用于界面开发、数据存储、日志记录和网络访问。核心功能设计、业务逻辑和作品内容由项目团队独立完成。

## 使用到的主要组件

### Qt

用于构建桌面界面、图表、数据库访问和网络请求。  
本项目主要使用了 Qt Widgets、Qt Charts、Qt Sql 和 Qt Network 模块。  
来源：[https://www.qt.io/](https://www.qt.io/)

### SQLite

用于本地保存课程、单元、学生和成绩数据。  
来源：[https://www.sqlite.org/](https://www.sqlite.org/)

### spdlog

用于程序运行日志和错误信息记录。  
来源：[https://github.com/gabime/spdlog](https://github.com/gabime/spdlog)

## 外部接口说明

### DeepSeek

系统支持调用 DeepSeek 接口生成教学分析建议。  
该接口属于外部服务，不属于本作品自研代码。  
来源：[https://platform.deepseek.com/](https://platform.deepseek.com/)

以下内容由项目团队自主设计和实现：
