EduStat 项目说明

一、项目简介
EduStat 是一个面向课堂教学场景的教学统计与课堂辅助系统，主要用于课程管理、学生信息维护、课堂点名、智能分组、成绩统计分析以及学期报告导出。

二、项目主要功能
1. 学科管理
   - 新增学科
   - 配置课程单元、权重与满分
   - 删除学科及关联数据

2. 学生信息管理
   - 新增、修改、删除学生信息
   - 导入学生名单与成绩
   - 导出学生信息与成绩
   - 按课程单元动态生成成绩表

3. 点名模块
   - 随机点名
   - 尽量不重复点名
   - 仅抽取未点到学生

4. 组队模块
   - 随机分组
   - 强配弱分组
   - 组队历史保存与恢复

5. 班级信息分析
   - 成绩段分布
   - 班级构成
   - 单元均分走势
   - 本地分析建议
   - DeepSeek 辅助分析

6. 导出功能
   - 导出学期报告
   - 导出课程成绩表
   - 按单元权重计算总分

三、开发环境
- 操作系统：Windows
- 开发语言：C++
- UI 框架：Qt Widgets
- 图表组件：Qt Charts
- 数据库：SQLite
- 网络请求：Qt Network
- 日志组件：spdlog
- 构建工具：CMake

四、主要文件说明
1. main.cpp
   程序入口，负责启动应用程序并创建主窗口。

2. mainwindow.h / mainwindow.cpp / mainwindow.ui
   项目核心文件，负责主界面、主要业务逻辑以及各功能模块之间的组织。

3. subjectcard.h / subjectcard.cpp
   学科管理页面中课程卡片的实现。

4. teamCard.h / teamCard.cpp
   组队页面中小组卡片的实现。

5. flowlayout.h / flowlayout.cpp
   自定义流式布局实现，用于卡片展示。

6. scoredelegate.h / scoredelegate.cpp
   成绩输入相关的代理与校验逻辑。

7. init.json
   项目配置文件，可用于配置外部分析接口相关参数。

8. dist
   辅助工具目录，包含部分导入导出转换工具。

9. README.md
   项目标准说明文档，包含更完整的项目介绍、构建方式和使用说明。

10. 开源代码与组件使用情况说明.md
    用于说明项目使用的开源框架、组件及相关授权情况。

11. 作品安装说明.md
    用于说明项目安装、运行与部署方式。

五、运行说明
1. 使用 Qt Creator 打开项目根目录下的 CMakeLists.txt
2. 选择 Qt 6.10.2 MinGW 64-bit Kit
3. 构建并运行项目
4. 如需发布 Release 版本，建议使用 windeployqt 补齐 Qt 运行依赖

六、补充说明
- 本项目为桌面端教学辅助系统，不替代学校统一教务平台
- 当前版本适合课堂演示、比赛展示与基础教学管理场景
- 若外部分析接口不可用，系统会自动回退到本地分析逻辑
